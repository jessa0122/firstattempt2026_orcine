# Orcine

AI-assisted web application project using Vue.js

---

## Framework

Vue.js (Vite)

## Module

Donation Module

---

## Project Overview

This project is an **Alumni Donation Dashboard Web Application** designed for Ateneo de Davao University. It allows alumni and administrators to:

* Register and login
* Select user roles (Alumni / Administrator)
* View donation dashboard
* Create donation campaigns
* Track donations and progress
* View leaderboard rankings
* Receive notifications

The system is built using **Vue.js** and developed with the assistance of multiple AI tools.

---

##  Installation Guide

Follow these steps to run the project on another computer:

```bash
git clone https://github.com/jessa0122/firstattempt2026_orcine.git
cd firstattempt2026_orcine
npm install
npm run dev
```

Then open:
http://localhost:5173

* generating UI and layout

## Prompt Used

**First Prompt:**
Create a Vue.js alumni donation dashboard web application with login, role selection, and admin panel.

**Main Prompt:**
Create a complete Vue.js web application that includes signup, forgot password, login, role selection, admin dashboard, donation campaigns, campaign creation, donation review, leaderboard, and notifications. Use a modern UI with blue and purple gradients and card layouts.
